В модуле dec.py  функция декоратор @time_this() для замера скорости выполнения декорируемой функции.
В параметр можно передать количество запусков декорируемой функции.

В модуле dec_m.py   определен класс декоратора Time_this аналогичного секундомера 
Для измерения скорости выполнения вашей функции используйте вызов класса декоратора с параметром
(количество прогонов вашей функции)
Например так:
@Time_this(20)
def my_function():
......
